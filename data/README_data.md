# Data Access

## Raw Sequencing Data

Raw shotgun metagenomic sequences will be deposited in NCBI SRA upon publication.

- **BioProject**: *To be assigned*
- **Platform**: Illumina (paired-end, 151 bp)
- **Kit**: TruSeq Nano DNA (350_META)
- **Samples**: 144 saliva samples

### Download sequences from NCBI (after publication)

```bash
# Install SRA Toolkit if needed
# conda install -c bioconda sra-tools

# Download all samples from the BioProject
prefetch PRJNAXXXXXX

# Convert to FASTQ format (paired-end)
fasterq-dump --split-files SRR*
```

## Sample Information

### Sample naming convention

Sample IDs follow the pattern: `PrefixNumber_SampleType`

**Control groups (healthy women):**
- `CTVM*` - Control group from Virgen Macarena
- `CTVMA*` - Control group from Virgen Macarena (alternative)
- `US*` - Control group from University of Seville

**Case groups (Sjögren syndrome):**
- `SSVM*` - Sjögren patients from Virgen Macarena
- `SVM*` - Sjögren patients (alternative naming)
- `SG*` - Sjögren patients (general)

### Metadata columns

| Column | Description |
|--------|-------------|
| sample_id | Unique sample identifier |
| group | Sjogren or Control |
| batch | Sequencing batch |
| collection_date | Sample collection date |
| age | Patient age at sampling |
| gender | Patient gender (all female) |

## Processed Data

### Species Abundance Table

The main output file `species_abundance.tsv` contains:

| Format | Description |
|--------|-------------|
| Rows | Bacterial species names |
| Columns | Sample IDs |
| Values | Relative abundance (fraction of total reads) |

This file is generated by `scripts/01_taxonomic_classification.sh` using Kraken2/Bracken.

### Example format

```
                          CTVM01_SSA  CTVM02_SSA  SSVM01_SSA  ...
Streptococcus salivarius  0.15234     0.12456     0.18234     ...
Rothia mucilaginosa       0.08123     0.09234     0.05123     ...
Veillonella parvula       0.06234     0.07345     0.04567     ...
...
```

## Database References

### Human Genome (for host removal)

- **Version**: GRCh38.p14 (GCF_000001405.40)
- **Source**: NCBI RefSeq
- **Purpose**: Remove human DNA contamination from saliva samples

```bash
# Download
wget https://ftp.ncbi.nlm.nih.gov/genomes/all/GCF/000/001/405/GCF_000001405.40_GRCh38.p14/GCF_000001405.40_GRCh38.p14_genomic.fna.gz
```

### Kraken2 Oral Microbiome Database

- **Type**: Custom database optimized for oral cavity
- **Content**: Bacterial genomes from oral microbiome
- **Purpose**: Taxonomic classification of non-host reads

The database can be built using:
```bash
kraken2-build --download-taxonomy --db oral_microbiome_db
kraken2-build --download-library bacteria --db oral_microbiome_db
kraken2-build --build --db oral_microbiome_db
```

Or use a pre-built oral microbiome database if available.

## Output Files Description

### From Preprocessing

| File | Description |
|------|-------------|
| `*_nonhost_1.fastq.gz` | Forward reads after host removal |
| `*_nonhost_2.fastq.gz` | Reverse reads after host removal |
| `fastp_summary.tsv` | Trimming statistics per sample |
| `host_removal_stats.tsv` | Host removal statistics |

### From Taxonomic Classification

| File | Description |
|------|-------------|
| `*.kreport` | Kraken2 report (taxonomic hierarchy) |
| `*.bracken` | Bracken abundance estimates |
| `species_abundance.tsv` | Combined species abundance matrix |
| `classification_summary.tsv` | Classification statistics |

### From Statistical Analysis

| File | Description |
|------|-------------|
| `Alpha_Diversity_Statistics.tsv` | Alpha diversity test results |
| `Beta_Diversity_Statistics.tsv` | PERMANOVA and betadisper results |
| `Differential_Abundance_Results.tsv` | Differentially abundant species |
| `Top20_Species_Summary_Statistics.tsv` | Summary of most abundant species |
